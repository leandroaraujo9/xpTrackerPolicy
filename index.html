Privacy Policy — Rubinot XP Tracker

Last updated: June 2026

Rubinot XP Tracker does not collect, transmit, or share any personal data. All data generated by the extension (XP snapshots, scan history, user preferences) is stored exclusively on the user's local device using the browser's built-in storage API and is never sent to any external server.

The extension uses the Chrome Identity API solely to verify that the signed-in Google account is authorized to use the extension. The email address is processed locally as a one-way hash and is never stored or transmitted.

The extension accesses rubinot.com.br only to read publicly available highscore data displayed on that website.
